"%" indicates your own variable.

MAKE A NEW REPO ONTO YOUR LOCAL!

1. git clone https://github.com/acfromspace/%repo_name%.git
2. cd %repo_name%
3. touch %some_file%
4. git add %some_file%
5. git commit -m "put your message here"
6. git push -u origin master (Now your master is updated and an official repo)

MAKE A LOCAL ONTO A NEW REPO!

1. Change computer directory to respective folder
2. git init
3. git remote add origin https://github.com/acfromspace/%repo_name%.git
4. git add . (adds all files)
5. git commit -m "put your message here"
6. git push -u origin master (Now your master is updated and an official repo)

Don't have any of the files and need to get them?
1. git clone https://gitlab.com/acdeleon/REPOSITORY_HERE.git

Make sure to have descriptions on ALL files.

Merging multiple repos:

Create new repo

git remote -v
# View current remotes

git remote add --fetch NAME https://github.com/user/repo.git

git merge -s ours --no-commit --allow-unrelated-histories NAME/master
# -s does short output

git read-tree --prefix=NAME/ -u NAME/master
# first name is where you name the folder in the main repo

git commit -m "Merge NAME into main_repo_name"

git pull -s subtree projectb master
# if more commits were made on the other side
