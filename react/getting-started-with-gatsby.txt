What is Gatsby?

Gatsby offers static-sites amazing speed using React
Gatsby also offers a pack that contains:
    Functional plugins
    Source plugins
    Transformer plugins

Gatsby Tutorial

Install a package manager
- Yarn is a package manager for Javascript
- NPM is also a package manager for Javascript
- yarn --version
- npm --version

Install node
- node --version

Install Gatsby's command line tool
- yarn global add gatsby-cli
- npm install --global gatsby-cli
- gatsby --version

Create a new site
- gatsby new NAME_OF_SITE && cd $_

Deploy a test server (localhost:8000)
- yarn dev (Fixed the script in package.json from develop to dev)
- gatsby develop

End deployment of a test server (localhost:8000)
- ctrl + c (Not sure why it's copy, but okay)

- yarn
